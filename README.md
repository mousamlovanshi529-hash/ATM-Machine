# ATM Machine Project

## Overview

This is a simple ATM Machine project developed using Python. It allows users to perform basic banking operations through a menu-driven program.

## Features

* PIN Authentication
* Check Balance
* Deposit Money
* Withdraw Money
* Mini Statement
* Exit

## Technologies Used

* Python 3
* Loops
* Conditional Statements
* Lists
* User Input

## How to Run

1. Save the file as `atm_machine.py`.
2. Open a terminal.
3. Run:

```bash
python atm_machine.py
```

## Default Details

* PIN: 1234
* Initial Balance: Rs. 100000

Learning Objectives

This project helps beginners understand:

*Python programming basics

*Menu-driven applications

*Looping and decision making

*Managing simple transaction records

*Building console-based applications
